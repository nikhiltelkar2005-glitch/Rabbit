🐰 Rabbit Project Roadmap

This document tracks the features we have added, what we are currently working on, and what is left to build. Updated as we make progress!

Phase 1: Authentication & Security ✅ Completed

  • College Email Verification — Only students with a college email can join
  • College Silos — Strict DB isolation; users only see their own college's content
  • Anonymous Identity Generation — Auto-assigned random names (e.g., "Curious Rabbit #4521")
  • Authentication System — Login, Signup, OTP, Password Reset
  • Rate Limiting — Brute-force protection on auth routes

Phase 2: Core Functionality ✅ Completed

  • Communities / Categories — Create and browse college communities
  • Anonymous Posting — Post without revealing real identity
  • Upvote / Downvote System — Reddit-style voting
  • Nested Comments & Replies — Threaded discussion
  • Search — Full-text search across posts and communities
  • Polls — 2–6 option polls with optional end date

Phase 3: College Vibe Features ✅ Completed

  • Campus Confessions — Via confession post flair
  • Ask Seniors / AMA — Time-limited Ask Me Anything sessions for placed students
  • Lost & Found — Via lost_found post flair
  • Post Flairs — placement, rant, advice, funny, confession, exam_help, lost_found, etc.
  • Media Uploads — Attach images to posts via Cloudinary
  • Event Hub — Clubs announce events with RSVP + capacity control

Phase 4: Safety & Moderation ✅ Completed

  • Report System — Report posts, comments, events, or users (7 reason types)
  • Admin Review Panel — pending → reviewed → action_taken / dismissed workflow
  • Ban / Unban Users — Admin can ban with a reason; banned users blocked at middleware
  • Admin Content Deletion — Delete any post or comment
  • Anti-Spam Rate Limits — Posts (5/hr), DMs (30/10min), OTP (3/10min), global (100/15min)
  • Admin Dashboard Stats — Real-time counts of pending reports and banned users

Phase 5: Gamification & Engagement ✅ Completed

  • Rabbit Karma — Earn/lose points from upvotes and downvotes
  • Achievement Badges — New Rabbit → Active Member → Top Contributor → Helpful Senior → Placement Guru
  • Trending Algorithm — Reddit-style hot score with time decay
  • Leaderboard — Top karma earners in college + your own rank
  • Anonymous DMs — Private messages by anonymous name, full inbox
  • Notifications — In-app alerts for replies, upvote milestones, badges, DMs, AMA answers

Future Backlog 💡 Ideas

  • Real-time Notifications — Socket.io WebSocket push instead of polling
  • College Marketplace — Sell books, calculators, hostel items anonymously
  • Study Material Exchange — Share notes, PDFs, previous year papers
  • Weekly Email Digest — Top 5 posts from your communities every Monday
  • Shareable Post Links — Unique URL per post for WhatsApp sharing
  • Edit / Delete own post — Within a 15-minute window
  • Saved / Bookmarked Posts — Save posts to read later
